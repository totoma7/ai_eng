# 4장. 실전 구성요소와 가드레일

Addy Osmani가 정리한 루프 엔지니어링의 **5대 구성요소 + 척추(외부 메모리)**, 그리고 프로덕션 루프에 필수인 가드레일.

## 4.1 5대 구성요소

### ① 자동화 (Automations) — 심장박동
- 주기적으로 도는 스케줄 태스크가 **일을 발견하고 트리아지**한다.
- 사람이 확인하러 가는 게 아니라, 작업이 **트리아지 인박스로 올라온다**.
- Claude Code / Codex의 `/loop`(주기 실행)과 `/goal`(조건 충족까지 실행)이 대표 구현.

### ② 워크트리 (Worktrees) — 병렬 격리
- git worktree로 작업 디렉토리를 분리해 **여러 에이전트가 충돌 없이 동시 작업**.
- 한 에이전트의 수정이 다른 에이전트의 체크아웃을 침범할 수 없다.

### ③ 스킬 (Skills) — 조직 지식
- `SKILL.md` 파일로 묶인 재사용 가능한 지시 패키지.
- 프로젝트 컨벤션, 빌드 절차, 컨텍스트를 인코딩 → **매 실행마다 같은 설명을 반복할 필요 제거**.

### ④ 플러그인 & 커넥터 — 환경 통합
- MCP 프로토콜 기반으로 이슈 트래커, DB, 스테이징 API, Slack 등 외부 시스템과 연동.
- 루프를 '제안만 하는 존재'에서 **실제 워크플로우 참여자**(PR 열기, 티켓 갱신, 알림 발송)로 전환.

### ⑤ 서브에이전트 — 검증 분리
- 역할이 다른 복수 에이전트: 하나는 탐색/구현, 다른 하나는 검증.
- **"자기 숙제 자기 채점" 방지**가 목적. 서브에이전트마다 다른 모델/노력 수준 배정 가능.

### 척추: 외부 메모리 (External Memory)
- 개별 대화 밖에 존재하는 마크다운 파일이나 Linear 보드.
- 완료된 작업과 다음 단계를 기록. **모델은 실행 사이에 잊어버리므로**, 영속 상태가 컨텍스트 재유도(re-derive) 비용을 막는다.
- Ralph Loop의 파일시스템 상태 유지와 같은 원리.

## 4.2 구체적 루프 아키텍처 예시 (Osmani)

```
아침 자동화 실행
  → 트리아지 스킬이 CI 실패, 열린 이슈, 최근 커밋을 읽음
  → 발견 사항을 영속 상태 파일에 기록
  → 실행 가능한 항목마다:
       격리된 worktree 생성 + 서브에이전트 2개 투입
       - 에이전트 A: 수정안 작성
       - 에이전트 B: 프로젝트 스킬/테스트 기준으로 리뷰
  → 커넥터가 PR 열고 티켓 갱신
  → 처리 못 한 항목은 트리아지 인박스로 → 사람이 리뷰
  → 다음 날 아침 실행은 이전 상태에서 이어감
```

## 4.3 종료 전략 — "LLM에는 '완료' 개념이 없다"

명시적 종료 조건이 없으면 루프는 토큰 예산을 무한히 태운다. 프로덕션 루프의 필수 장치:

| 장치 | 내용 |
|------|------|
| **반복 상한 (hard cap)** | 최대 사이클 수 제한 |
| **토큰/달러 예산** | 실행 전에 설정 |
| **무진행 감지** | 반복 간 상태가 안 변하면 탈출 |
| **자동 검증 기준** | 테스트 통과, 커버리지 임계값, 타입 체크 |
| **타임아웃** | 도구 호출 단위 + 태스크 단위 |
| **서킷 브레이커** | 도구 재시도 폭주 차단 |
| **휴먼 체크포인트** | 비가역적 행동(머지, 배포, 삭제) 전 사람 승인 |

> "목표는 자율성을 없애는 게 아니라 **경계 짓는(bound)** 것이다."

## 4.4 실패 모드 카탈로그

| 실패 모드 | 원인 | 대책 |
|----------|------|------|
| 무한 루프 | 객관적 검증 부재 — 멈출 조건 없이 계속 다듬음 | 평가자 분리 + 반복 상한 |
| 목표 표류 (goal drift) | 모호한 명세 → 곁가지 목표 추구 | 목표를 검증 가능하게 명세 |
| 컨텍스트 오버플로 | 긴 세션이 추론 품질 저하 | Ralph식 컨텍스트 리셋 + 외부 메모리 |
| 침묵 실패 | 도구는 실행되나 실제 진전 없음 | 무진행 감지, 결과 기반 검증 |
| 토큰 폭발 | 중복 호출 폭주 (사례: 5분에 400회) | 예산 + 서킷 브레이커 |
